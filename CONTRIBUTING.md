# 🤝 Contributing Guide

Thanks for your interest in contributing to this project — you're awesome! 🚀

We welcome all kinds of contributions, big or small. Whether it's fixing a typo, reporting a bug, or suggesting a whole new feature, you're helping make this project better for everyone. 🙌

---

## 🛠️ How to Contribute

### 1. Fork the Repo 🍴

Click the **Fork** button at the top right of this page to create your own copy of the repo.

### 2. Clone Your Fork

```bash
git clone https://github.com/udiram/SpectrePDF.git
cd SpectrePDF
````

### 3. Create a Branch

Use a descriptive name for your branch:

```bash
git checkout -b fix/issue-123
```

### 4. Make Your Changes

Write some great code, squash that bug, improve the docs — whatever you're here for!

### 5. Commit Like a Pro

Write clear, concise commit messages:

```bash
git commit -m "fix: correct off-by-one error in pagination"
```

### 6. Push and Open a PR

```bash
git push origin your-branch-name
```

Then go to the GitHub repo and open a **Pull Request** (PR). Please describe **what** you changed and **why**.
